Web 3D ROS Visualisation

We assisted TORK to bring online a 3D ROS visualisation application that uses sensor data and allows to manipulate the model from the client.

Premise gathers and analyzes millions of observations captured daily by a global network of contributors. DRM for Live Content Streaming

In this project we implemented a DRM for protection of live content streaming and dynamic traitor detection and revocation. State of the Art encryption similar to AACS (Blu-ray).

Real-time Transcription through Crowdsourcing

Scribe is a research project led by researcher Jeff Bigham that collects audio from a mobile device and schedules workers to perform captioning on small audio segments.

The distributed captions are then merged into a single text, returning a consolidated transcript to the mobile user.

Live online video streaming

For this project we created new content protection solutions for live video delivered over the Internet, using Adobe Media Server instances running in the Amazon Cloud and streaming content via HDS (HTTP Dynamic Streaming).

The backend processing is implemented in Python and uses MongoDB for persistence.
