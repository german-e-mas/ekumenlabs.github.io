Autonomous Android-based vehicles

The aim of this project is to develop an autonomous robot system using ROS that executes entirely on an Android device.

We contributed to ROS Java by implementing a base controller from scratch, extend it to use the multiple radios instead of just WiFi and added the ability to create externally reachable masters in an Android device.

After a first success with ground vehicles navigation we adapted the web interface for flying drones, helped creating the communication system between the interface and the drone and adapting the Pixhawk onboard computer project to that system. We also developed an autonomous mission teleoperation App (Android) for flying vehicles with live video feed and waypoint support.
